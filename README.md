# CookinTolerance

CookinTolerance es una aplicación móvil desarrollada en Flutter para la búsqueda y consulta de recetas de cocina, orientada a personas con intolerancias alimentarias.  
La aplicación permite filtrar recetas según distintas intolerancias y consultar información detallada de cada plato de forma sencilla.

Este proyecto se desarrolló como proyecto final de un curso de Flutter y posteriormente se ha revisado y mejorado como proyecto personal para mantener actividad reciente en desarrollo móvil.

---

## Funcionalidades

- Listado y búsqueda de recetas
- Visualización del detalle de cada receta
- Filtrado por intolerancias alimentarias
- Gestión de recetas favoritas
- Persistencia de datos en local
- Manejo de estados de carga y error

---

## Tecnologías utilizadas

- Flutter / Dart
- Gestión de estado mediante una solución sencilla orientada a la UI
- Persistencia local de datos
- Estructura basada en separación de responsabilidades
- Servicios desacoplados para la obtención y gestión de datos

## Instalación y ejecución

1. Clonar el repositorio:
   git clone https://github.com/Mcaballe1995/CookinTolerance.git

2. Instalar dependencias:
   flutter pub get

3. Ejecutar la aplicación:
   flutter run
